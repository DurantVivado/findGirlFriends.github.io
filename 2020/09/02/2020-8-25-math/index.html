public static long gcd(long a, long b){
    if (b == 0){
        return a;
    }
    return gcd(b, a % b);
}

快速幂

https://blog.csdn.net/qq_19782019/article/details/85621386

我们思考一个问题，求$2^{10E9}$的最后三位数。

  1. (a + b) % p = (a % p + b % p) % p （1）
  2. (a - b) % p = (a % p - b % p ) % p （2）
  3. (a * b) % p = (a % p * b % p) % p （3）

数学上可以证明只要我们每一步都取后三位，那么最后的结果就是答案。

快速幂的核心算法是【分治】，即把每一次幂都分为两半。

  • 如果幂为偶数则，将幂除2，将基乘2，并且取余数。
  • 如果幂为奇数，将幂减1除2，多出的一次幂乘到结果中，并且取余数。

如果最后幂为0，表示计算完毕。

我们可以适当进行优化

  • a&1==true表示a为奇数，因为偶数二进制最后一位一定是0.
  • 幂减1除2，可以直接省略，因为/2运算会忽略小数点。同时>>1也表示除2.
long long fastPower(long long base, long long power, int digits) {
    //digits表示取最后几位
    long long result = 1;
    while (power > 0) {
        if (power & 1) {//此处等价于if(power%2==1)
            result = result * base % digits;
        }
        power >>= 1;//此处等价于power=power/2
        base = (base * base) % digits;
    }
    return result;
}

在我们分解为（底部情况）三平方数之前，还有两种特例需要检查：

  1. 该数本身是一个完全平方数 即n=(int)(sqrt(n)*sqrt(n))

  2. 如果一个数可以分解为两个平方数之和，不幸的是，我们只能通过枚举来找到这样的数。

算法：

  1. 检查该数是否为完全平方数

  2. 如果该数可以表示为$n == 4^k(8m+7)$，那么可以确切的说，答案肯定为4；

while(n%4==0) n /= 4;
if((n-7)%8==0) res = 4;
  1. 我们看该数，是否能由更少的完全平方数组成。
for(int k = n-1;k >= 1;k--)
{
    if(isPerfectSquare(n*n-k*k)) res = 2;
}

代码：

class Solution {
public:
    bool isPerfectSquare(int n)
    {
        return n == ((int)sqrt(n))*((int)sqrt(n));
    }
    int numSquares(int n) {
    //一些很特殊的情况12 = 2^2+2^2+2^2; 18 = 3^2+3^2;1000= 30^2+10^2;61=5^2+6^2
    if(isPerfectSquare(n)) return 1;
    int nn = n;
    //判断是否为四平方数
    while(nn%4==0) nn/=4;
    if((nn-7)%8==0) return 4;
    for(int k = n-1;k >= 1;k--)
    if(isPerfectSquare(k))
    {
        // printf("%d:\n",k);
        if(isPerfectSquare(n-k)) return 2;
    }
    return 3;

    }
};

复杂度分析

  • 时间复杂度： 最好的情况下是$O(1)$，比如$n$是完全平方数, 最坏$O(N)$，即该数有两个平方数组成。
  • 空间复杂度: $O(1)$，常数级别空间占用。
